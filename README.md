## FULLSTACK ASSIGNMENT

https://fullstack-assignment-app.herokuapp.com/

## How to use

#### Endpoints:

`http://localhost:5000/api/events`
`api/events`

#### To local run:

backend start:

`yarn install`
`yarn server`

frontend start:

`cd client`
`yarn install`
`yarn start`
